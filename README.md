# Beauty Salon Website

A fully responsive, multi-section website designed for a professional beauty salon. This project focuses on high-quality UI/UX and clean code structure.

## Features
* Fully Responsive Design: Optimized for all screen sizes, from mobile to desktop.
* Visual Hierarchy: Focused on consistent styling and optimized page flow to improve user experience.
* Multi-section Layout: Includes dedicated sections for services, pricing, gallery, and contact information.

## Technologies Used
* HTML5
* CSS
* Clean Layout Principles
